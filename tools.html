Tools I Enjoy

Writing / Notes

HackMD is one of my favorite tools for collaborative Markdown writing. I like it for shared notes, writeups, and documents where multiple people need to edit or comment together.

Obsidian is what I enjoy using for personal notes. I like that it keeps notes organized locally and makes it easy to connect ideas over time.

MacOS Apps

Rectangle is my go-to app for window snapping on Mac. It makes arranging windows much faster and helps keep my workspace organized.

Tabby is my favorite terminal. I like using it as a cleaner and more customizable terminal setup.

Research

Typst is a tool I enjoy for writing papers and technical documents. I like that it feels lightweight while still being good for polished research writing.
